ReactorSim - Prototype (Educational)
-----------------------------------
Nội dung:
- index.html : single-file client-side simulation (HTML + CSS + JS)

Hướng dẫn nhanh:
- Giải nén reactor_sim.zip và mở file index.html trên trình duyệt để thử.
- Upload index.html vào public_html trên cPanel hoặc lên GitHub Pages để host.
- Không sử dụng để học vận hành thiết bị thực tế. Đây là mô phỏng giải trí/giáo dục.

Muốn nâng cấp:
- Thêm WebSocket/Node.js backend cho multiplayer/leaderboard.
- Lưu events vào DB để replay.
- Thêm sound FX, animation, tutorial từng bước.

Bản quyền và disclaimers:
- Prototype được tạo theo yêu cầu. Mọi thông tin mô phỏng là ngắn gọn, phi thực tế và KHÔNG cung cấp hướng dẫn vận hành thực tế.
